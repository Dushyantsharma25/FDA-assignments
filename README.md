## Hey there , if you want to run codes of this repo,
you can simply run this command for .py 

for linux and mac :- 

    python3 file.py

for windows :-

    python file.py

and for .cpp , first you have to compile the using g++ compiler :-

    g++ compiled_file -o file.cpp

now you can run it using :-

    ./compiled_file
